# TrainFlow
TrainFlow is a full - stack workout planning application that allows users to browse exercises, create workouts, and organize exercise routines.

## Features

- User Authentication (register/login)
- Exercise Library
- Workout Creation and Managament
- Add Exercises to workouts
- Dashboard for Navigating the App

## Tech Stack

Frontend
- React
- React Router
- Axios

Backend
- Node.js
- Express
- MongoDB
- Mongoose
